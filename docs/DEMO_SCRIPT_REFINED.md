## üé¨ **Production Tips**

### **Critical Moments (Must Get Right)**

1. **Citation Click (Scenes 5, 6, 9) - MOST IMPORTANT**
   - These are your trust moments
   - Make citation editor, file open, and widget citation badge **crystal clear**
   - Use zoom/close-up on citation badges
   - Show the exact text match in the PDF

2. **Speed Up Upload (Scene 2)**
   - Upload progress bar can be 2x-4x speed
   - Don't make viewers wait for real upload time
   - Keep it smooth, not jarring

3. **Widget Demo (Scenes 8, 9, 10)**
   - Use a real company website mockup (not generic placeholder)
   - Make widget feel native and integrated
   - Show smooth animations (widget opening, typing indicator)
   - Highlight citation badge clearly

4. **File Open Highlight (Scene 6)**
   - Show exact text match highlighted in PDF
   - Use zoom to make it obvious
   - This is proof of trust - make it undeniable

5. **Music**
   - Subtle, driving background track
   - No vocals (distracts from voiceover)
   - Energetic but not overwhelming
   - Fade out during critical moments (citation clicks)

6. **Mouse Halo**
   - Use tool to highlight cursor (yellow halo or similar)
   - Makes it easy to follow action
   - Essential for citation clicks and button presses

### **Technical Setup**

- **Screen Resolution:** Record at 1920x1080 or higher
- **Zoom Level:** Zoom in slightly (don't show full 4K monitor)
- **Frame Rate:** 30fps minimum (60fps preferred for smoothness)
- **Audio:** Clear voiceover, no background noise
- **Format:** MP4 (H.264) for compatibility

### **Editing Workflow**

1. **Record Screen Actions** (using Loom, Camtasia, or OBS)
2. **Import to Canva Video Editor** (or preferred editor)
3. **Add Voiceover Track** (based on script)
4. **Add Transitions** (smooth, professional)
5. **Add Music** (subtle background track)
6. **Add Mouse Halo** (if not recorded live)
7. **Speed Up Boring Bits** (upload progress, loading)
8. **Final Export** (1080p minimum, 4K preferred)
